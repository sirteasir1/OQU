# OQU
Это образовательная платформа на базе искусственного интеллекта, которая генерирует персонализированные уроки и тесты по математике на основе интересов студентов. Создана для казахстанских школьников (7-11 классы) и превращает абстрактные математические концепции в понятный и увлекательный контент.
